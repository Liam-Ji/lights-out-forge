# 项目总览

## 目标

在展厅展示“AI 自动化开发剧场”：观众输入一句业务需求，系统自动生成可点击、可演示、有业务结果输出的原型。

当前阶段只验证一条主链路：

> 业务需求 → AI 生成原型 → 浏览器预览 → 业务结果展示

## 当前结论

| 项目 | 结论 |
|---|---|
| 目标观众 | 领导、客户、技术决策者、行业专家 |
| 主场景 | 园区视频事件检索与报告生成 |
| 一句话需求 | 请帮我做一个园区视频事件检索看板，支持查询昨晚是否有人员进入重点区域，并自动生成事件摘要报告。 |
| 第一阶段技术选型 | 鸑鷟 AI + 类 OpenClaw / 自定义交互程序 + Codex + 在线大模型 API + 脱敏样例数据 |
| 辅助技术路线 | 类 OpenClaw / 自定义交互程序 + Codex 用于不限技术栈的自由原型生成，Open Design 用于视觉探索，OpenHands 用于代码和插件开发辅助 |
| 技术路线 | 以鸑鷟为主线验证业务原型生成，私有能力通过能力清单和后续插件/工具接入 |
