# 黑灯工厂项目总览

## 目标

在展厅展示“AI 自动化开发剧场”：观众输入一句业务需求，AI 基于鸑鷟框架和鸑鷟已有能力，快速生成一个可点击、可演示、有业务结果输出的原型系统。

本项目当前不是建设完整平台，也不是马上实现所有技术细节。当前阶段只做三件事：

1. 定清楚第一版演示场景。
2. 明确场景为什么适合展厅。
3. 规划后续实施路线。

## 当前主场景

**园区视频事件检索与报告生成**

一句话需求：

> 做一个视频事件检索看板，输入“昨晚园区是否有人员进入重点区域”，系统自动检索事件、展示截图，并生成事件摘要报告。

## 为什么选这个场景

- 观众容易理解：一句话查视频事件，结果可视化明确。
- 能体现鸑鷟能力：视频语义搜索、视频播放、报告生成、大屏展示。
- 演示链路短：不依赖实时摄像头、实时视频流、地图和工单系统。
- 稳定性高：第一版可以使用 mock/脱敏样例数据。
- 可扩展：后续可升级到园区安防、巡检/巡防或应急指挥。
